# Knowledge Center

Bienvenue dans ton **Knowledge Center** — un espace structuré, modulaire et durable regroupant l’ensemble de tes connaissances professionnelles : SEO, performance, architecture front-end, guidelines, bonnes pratiques et documents transverses.

👉 **Pour accéder directement à la liste complète des fichiers :**  
➡️ [Voir l’index global](./index.md)

## 🎯 Objectifs du Knowledge Center

- centraliser les connaissances
- éviter la duplication
- structurer l’information par domaine
- créer une base durable et évolutive
- faciliter la navigation et la réutilisation
- préparer une future migration vers un site MDX si nécessaire

## 📁 Structure du KC

```

/
seo/
README.md

...

```

## 🧭 Navigation

- SEO → stratégie, contenu, structure, maillage interne
- Performance → rendu, JS, images, caching, Core Web Vitals
- Architecture (optionnel) → patterns, conventions, décisions techniques
- Guidelines (optionnel) → rédaction, design system, naming, conventions

## 🧱 Philosophie

- **Un dossier = un domaine**
- **Un README = l’introduction**
- **Un index = la navigation**
- **Aucun doublon entre SEO et Performance**
- **Des cross‑links explicites**
- **Des fichiers courts, autonomes, lisibles**
- **Une architecture prête pour MDX**
